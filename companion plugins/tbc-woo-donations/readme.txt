Complete donation product system for WooCommerce — name-your-price, suggested amounts, deposits, fee recovery, donor wall, and more.

== Description ==

TBC WooCommerce Donations turns WooCommerce into a full donation platform: name-your-price fields, suggested amount presets, recurring / one-time toggles, deposit-style pledges, fee recovery (let donors cover Stripe's cut), and a public donor wall. Ships as a WooCommerce product type so donations flow through the normal WC order pipeline.

== Installation ==

1. Install and activate WooCommerce 9.0+.
2. Upload the `tbc-woo-donations` folder to `/wp-content/plugins/`.
3. Activate the plugin through the Plugins menu.
4. Create a new product of type **Donation** and configure options.
