Manage member perks for WooCommerce Subscriptions with renewal-based discounts and role management.

== Description ==

TBC Member Perks adds renewal-based discounts and automatic role assignment to WooCommerce Subscriptions. Customers earn growing discounts as they renew; their WordPress role updates to match their subscription tier. Useful for tiered membership programs where long-time members get better pricing and more access.

== Installation ==

1. Install and activate WooCommerce Subscriptions.
2. Upload the `tbc-member-perks` folder to `/wp-content/plugins/`.
3. Activate the plugin through the Plugins menu.
4. Configure perk tiers from the admin settings page.
