New member tracking dashboard with community stats. Fluent Community native.

== Description ==

TBC New Members renders a front-end dashboard showing recent signups, community growth over time, and member activity stats. Designed for Fluent Community staff to surface engagement metrics to moderators or volunteer onboarding teams. Dashboard is embedded via the `[members-dashboard]` shortcode and gated by the `edit_posts` capability.

== Installation ==

1. Install and activate Fluent Community.
2. Upload the `tbc-new-members` folder to `/wp-content/plugins/`.
3. Activate the plugin through the Plugins menu.
4. Place the `[members-dashboard]` shortcode on a page accessible to your moderator roles.
