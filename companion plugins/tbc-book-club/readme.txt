Manages and displays book club audiobooks with chapter support, progress tracking, and bookmarks.

== Description ==

TBC Book Club Manager lets community admins publish audiobooks with chapter playlists and track listener progress. Members can bookmark spots in a chapter and resume later from any device. Integrates with the TBC Community App for in-app listening; data flows through REST endpoints consumed by the mobile client.

== Installation ==

1. Upload the `tbc-book-club` folder to `/wp-content/plugins/`.
2. Activate the plugin through the Plugins menu.
3. Configure books and chapters from the admin menu.
