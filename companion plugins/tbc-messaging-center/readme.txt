Unified messaging system for SMS, calls, voicemail, and scheduling with Fluent Community integration.

== Description ==

TBC Messaging Center wires Twilio SMS, voice calls, voicemail, and scheduled messaging into the WordPress admin, with a Fluent Community integration layer so community staff can reach members via text. Includes a scheduler for recurring SMS campaigns and media attachments for MMS.

== Installation ==

1. Upload the `tbc-messaging-center` folder to `/wp-content/plugins/`.
2. Activate the plugin through the Plugins menu.
3. Configure Twilio credentials (SID, Token, Messaging Service SID) from the admin settings page.
