Event calendar system — sorts WooCommerce products by event date and displays them in a calendar view.

== Description ==

TBC WooCommerce Calendar turns WooCommerce products into calendar events by adding event date, time, location, RSVP, and recurrence metadata. Products with event dates render in a calendar shortcode view and can be subscribed to as an ICS feed. Designed for community sites running Fluent Community + WooCommerce where each "event" is a sellable product.

== Installation ==

1. Install and activate WooCommerce.
2. Upload the `tbc-calendar-fluent` folder to `/wp-content/plugins/`.
3. Activate the plugin through the Plugins menu.
4. Edit any WooCommerce product and fill in the event date / time / location fields.
5. Use the `[tbc-calendar]` shortcode on a page to render the calendar view.
