About OTA Updates

    

      
Push JavaScript-layer changes directly to your users without a store build. Updates download silently and apply on the next app launch.

      

        

          Works with OTA
          
    
            
  • Custom modules & screens
    • 
            
  • Styling & theme changes
    • 
            
  • Config changes
    • 
            
  • Images & assets
    • 
            
  • Bug fixes in JS/TS code
    • 
          

        

        

          Requires Store Build
          
    
            
  • New native modules
    • 
            
  • Permission changes
    • 
            
  • Expo SDK upgrades
    • 
            
  • Native config in app.json
    •

Update History

      
    

    

      
OTA updates only reach devices on the matching runtime version. Once you release a new native build (e.g. v3.4.0), old updates (e.g. v3.3.0) will no longer be delivered and can be safely removed.
