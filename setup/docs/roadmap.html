Not Applicable to Mobile

These Fluent Community features are web-specific and intentionally not implemented in the native app:

  • Custom CSS / JS injection — web portal only
  • Sidebar links — mobile has no sidebar (uses tab bar + menu)
  • Rich post layout toggle — mobile uses native card layout
  • Member list layout toggle — mobile uses native list layout
  • "Powered by" branding — not present in native app
  • SEO sitemap — web indexing, not relevant for app
  • Portal slug configuration — web URL path, app uses deep link mapping
