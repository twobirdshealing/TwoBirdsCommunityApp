Adds FluentCommunity custom profile field triggers to Uncanny Automator — fire automations when a user updates a specific custom field to a specific value.

== Description ==

This add-on registers new triggers in Uncanny Automator that fire when a user updates a FluentCommunity custom profile field to a specific value. Use it to automate role changes, send notifications, tag users, or kick off any other Automator action in response to profile updates.

== Installation ==

1. Install and activate Uncanny Automator and FluentCommunity.
2. Upload the `tbc-automator-fcom-fields` folder to `/wp-content/plugins/`.
3. Activate the plugin through the Plugins menu.
4. The new triggers appear under the **TBCAUTOMATIONS** integration in Uncanny Automator.
