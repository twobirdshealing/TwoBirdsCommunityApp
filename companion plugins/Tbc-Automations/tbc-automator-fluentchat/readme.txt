Adds a "Send Direct Message" action to Uncanny Automator — sends direct messages via FluentChat when automations run.

== Description ==

This add-on registers a "Send Direct Message" action in Uncanny Automator that sends a FluentChat DM from one user to another when an automation fires. Useful for welcome flows, onboarding sequences, or any scenario where you want to reach out directly in response to a trigger.

== Installation ==

1. Install and activate Uncanny Automator and FluentChat.
2. Upload the `tbc-automator-fluentchat` folder to `/wp-content/plugins/`.
3. Activate the plugin through the Plugins menu.
4. The new action appears under the **TBCAUTOMATIONS** integration in Uncanny Automator.
