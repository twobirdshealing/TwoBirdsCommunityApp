Role-based messaging restrictions for Fluent Community. Control which WordPress roles can initiate DMs and send community chat messages.

== Description ==

TBC Message Roles adds role-based gating to Fluent Community's messaging. Site admins can specify which WordPress roles are allowed to start direct messages and which can send messages in community chat spaces. Useful for read-only member tiers, moderation policies, or membership-based access control.

== Installation ==

1. Install and activate Fluent Community.
2. Upload the `tbc-message-roles` folder to `/wp-content/plugins/`.
3. Activate the plugin through the Plugins menu.
4. Configure allowed roles from the admin settings page.
