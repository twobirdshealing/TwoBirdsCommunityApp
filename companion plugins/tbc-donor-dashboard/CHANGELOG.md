# Changelog

All notable changes to the Two Birds Church Donor Dashboard plugin.

## 1.2.05

- Converted all hardcoded CSS colors to Fluent Community `--fcom-*` CSS variables with fallback defaults
- Added dark mode support via `html.dark` selector (syncs with Fluent Community theme)
- Error and success message colors adapted for dark backgrounds
- Button styling aligned with Fluent Community button tokens (`--fcom-primary-button`)
- Button hover uses `filter: brightness(1.1)` pattern (matches fluent-starter theme)
- Box shadow adjusted for dark mode visibility
