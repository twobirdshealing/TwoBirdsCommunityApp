Display WooCommerce event products and attendee management.

== Description ==

TBC Participant Frontend adds attendee-management UI for WooCommerce event products — attendance lists, opt-in/opt-out tracking, event chat spaces, and bulk SMS to attendees (via the TBC Messaging Center plugin when installed). Designed for hosts running paid events through WooCommerce who need a lightweight roster and communication tool.

== Installation ==

1. Install and activate WooCommerce.
2. (Optional) Install TBC Messaging Center for SMS features.
3. Upload the `tbc-participant-frontend` folder to `/wp-content/plugins/`.
4. Activate the plugin through the Plugins menu.
