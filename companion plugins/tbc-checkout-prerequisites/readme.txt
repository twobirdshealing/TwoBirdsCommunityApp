Displays prerequisite steps customers must complete before checkout.

== Description ==

TBC Checkout Prerequisites gates WooCommerce checkout behind a configurable list of steps — for example, completing a Gravity Forms application or finishing a LearnDash course. Customers see a clear dashboard of remaining steps on the checkout page; staff can review and approve submissions from the admin side.

== Installation ==

1. Upload the `tbc-checkout-prerequisites` folder to `/wp-content/plugins/`.
2. Activate the plugin through the Plugins menu.
3. Configure prerequisite steps from the admin settings page.
