Adds a "Send Notification" action to Uncanny Automator — sends native FluentCommunity in-app notifications when automations run.

== Description ==

This add-on registers a "Send Notification" action in Uncanny Automator that fires a native FluentCommunity in-app notification to a target user. Supports routing to space home, a post, a community post, user profile, course home, or course lesson — all with auto-resolved slugs.

== Installation ==

1. Install and activate Uncanny Automator and FluentCommunity.
2. Upload the `tbc-automator-fluent-notification` folder to `/wp-content/plugins/`.
3. Activate the plugin through the Plugins menu.
4. The new action appears under the **TBCAUTOMATIONS** integration in Uncanny Automator.
